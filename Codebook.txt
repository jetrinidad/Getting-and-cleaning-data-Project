# Data Code Book


* The Tidy Dataset contains the data of the mean and std values of the observations from the training and testing data. The tidy dataset contains 10299 observations and 68 variables in total
* The Summarized Dataset contains the averaged values of each variable per subject and activity. The data set 180 (rows) and 68 (columns)

## Activities
### The following activities were performed by 30 volunteer subjects to produce the train and test data
*LAYING
*SITTING
*STANDING
*WALKING
*WALKING_DOWNSTAIRS
*WALKING_UPSTAIRS

## Variable Units

subjectnumber is the identifier used to denote each experiment volunteer.
activityname denotes the specific activity to be measured in the experiment.
The rest the variables are the readings from the gyroscope and accelerometer.
